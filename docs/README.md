# Documentation (X-OCTANE Artifact)

This `docs/` folder contains detailed documentation for reproducing the X-OCTANE experimental results and figures.

## Contents

- `README.md` (this file)  
  Overview of documentation and where to start.

- `REPRODUCIBILITY.md`  
  Step-by-step instructions to run the full pipeline, including environment setup, dataset configuration, and expected outputs.

- `DATA.md`  
  Dataset directory layout and required data fields/metadata expected by the scripts.

- `FIGURE_INDEX.md`  
  Mapping of paper figures/tables to generated output filenames and the script(s) that produce them.
